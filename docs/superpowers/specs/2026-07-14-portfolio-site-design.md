# Персональный сайт-портфолио — дизайн

## Контекст

Референс: https://zarazaex.xyz/ — минималистичная одностраничная визитка на моноширинном шрифте, светлая тема, подчёркнутые ссылки, короткие разделы вида `name(описание): ссылка`.

Цель: сделать аналог в том же минималистичном духе, но заточенный под поиск работы — портфолио/резюме на основе `Шамсутдинов_Марсель_Резюме.pdf`, а не хакерская визитка.

## Технический стек

- Статический сайт: `index.html` + `style.css` + `script.js`. Без сборки, без зависимостей.
- Хостинг: GitHub Pages (репозиторий `marselnet86.github.io` или project pages в существующем аккаунте `github.com/MarselNet86`). Публикация в GitHub Pages — отдельный шаг, требует явного подтверждения пользователя перед пушем.
- Шрифт: моноширинный (`JetBrains Mono` / `Fira Code`, фолбэк на системный `monospace`).
- Цвет: монохромная палитра (чёрный/белый/оттенки серого), без ярких акцентов. Подчёркнутые ссылки, как в оригинале.

## Тема (светлая/тёмная)

- По умолчанию — авто по `prefers-color-scheme`.
- Ручной переключатель тем в хедере (иконка/текст, как `old/new` в оригинале).
- Выбор пользователя сохраняется в `localStorage` и переопределяет системную настройку при следующем визите.
- Реализация через CSS-переменные + атрибут `data-theme` на `<html>`.

## Язык (ru/en)

- Переключатель `ru/en` в хедере, как в оригинале.
- Весь текстовый контент дублируется в разметке (пара элементов с `data-lang="ru"` / `data-lang="en"`, JS переключает `display` и атрибут `lang` на `<html>`).
- Выбор языка сохраняется в `localStorage`.
- По умолчанию — русский.

## Тестирование / проверка

- Открыть `index.html` в браузере (Browser pane), проверить:
  - Переключение темы (светлая/тёмная), сохранение выбора после перезагрузки
  - Переключение языка (ru/en), сохранение выбора после перезагрузки
  - Все ссылки на проекты и контакты кликабельны и ведут на верные адреса
  - Скачивание PDF резюме работает
  - Адаптивность на мобильной ширине (375px)
- Ручная проверка орфографии/содержания обеих языковых версий
